# E-Commerce Sales Analysis Project

## Overview

This project performs a complete data analysis on an e-commerce sales dataset. It includes data cleaning, analysis, and visualization to extract meaningful business insights.

---

## Objectives

* Analyze product-wise sales performance
* Identify top-performing regions
* Study monthly sales trends
* Visualize data using charts
* Generate actionable insights

---

## Dataset Description

The dataset contains the following columns:

* **Date** → Transaction date
* **Product** → Product name
* **Quantity** → Units sold
* **Price** → Price per unit
* **Customer_ID** → Unique customer identifier
* **Region** → Sales region
* **Total_Sales** → Total revenue per transaction

---

## Technologies Used

* **Python**
* **Pandas** (Data Analysis)
* **Matplotlib** (Data Visualization)

## How to Run the Project

### 1. Clone the repository

```
git clone <your-repo-link>
cd e-commerce-sales-analysis
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the project

```
python main.py
```
